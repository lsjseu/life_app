## 1. 设计系统

### 1.1 设计理念

**核心设计原则**：
- **简洁清晰**：4Tab导航，功能分工明确
- **自然交互**：语音、拍照等自然交互方式优先
- **温暖关怀**：健康类产品需要传递温暖和关怀
- **高效便捷**：减少操作步骤，提升使用效率

**设计关键词**：
- 清新
- 温暖
- 专业
- 易用

|---------|---------|
| < 375px | 小尺寸模式，间距缩小 |
| 375px - 414px | 标准尺寸 |
